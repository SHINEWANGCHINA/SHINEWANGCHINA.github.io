在用element-ui和antv G6 画canvas自定义图形的时候，遇到一个问题。
在 InputNumber 中需要传index值，InputNumber 官方是只能传当前值。
我试着传index值的时候，然后用form数据来传当前值，但是发现当前值永远是上一个值。无法获取准确的当前值。

element-ui 组件 InputNumber

官方是用 @change 来跟踪计数器的值
